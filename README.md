# FPGA Projects

Digital design projects built on the iCEBreaker FPGA (Lattice iCE40UP5K) using the open-source Yosys + nextpnr synthesis flow. Built as a self-study sprint toward a chip design career.

**Background:** ECE undergraduate at The Ohio State University. Co-founder of Buckeye Silicon, OSU's chip design student org.

## Toolchain

- **Simulation:** Icarus Verilog, Verilator
- **Synthesis & PnR:** Yosys, nextpnr-ice40
- **Programming:** icestorm (iceprog)
- **Waveforms:** GTKWave
